Week 9

Arduino and electronics


This week, we learned all about Arduino. It is microcontroller platform that is easy to use for electronics projects.

It allows us to connect electronics through its pins so we can control things.

We finished every single project on the Sparkfun Inventors Kit Guide book, it took quite long but it is interesting to see how Arduino works.

We did projects like blink LED, photoresistor, buzzer, the game simon says, distance sensor, LCD display screen, motor and even an autonomous robot at last
